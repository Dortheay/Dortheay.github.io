Publications

FSE 2026

TestTailor: Generating High-Coverage Tests via Path-Proximal Tests with LLMs

Xiaoxuan Zhou, Yiling Lou, Jinhao Dong, Dan Hao

TestTailor transforms vague coverage goals into precise path-level guidance for LLM-based test generation by combining path-proximal test analysis with symbolic constraints from uncovered paths. On the CODAMOSA benchmark, it achieves up to 12.78% higher coverage while reducing API cost by ~60%.

Software Testing LLMs Test Generation
ICASSP 2026

Code-Vision: Evaluating Multimodal LLMs Logic Understanding and Reasoning Capabilities Through Code Generation

Xiaoxuan Zhou et al.

Code-Vision is a novel benchmark for evaluating the logical understanding and reasoning capabilities of multimodal LLMs in code generation tasks. It uses flowcharts as primary input, making visual comprehension essential for generating correct code.

Multimodal LLMs Code Generation Benchmark

Boosting Text-To-Image Generation via Multilingual Prompting in Large Multimodal Models

Yongyu Mu, Hengyu Li, Junxin Wang, Xiaoxuan Zhou, Chenglong Wang, Yingfeng Luo, Qiaozhi He, Tong Xiao, Guocheng Chen, Jingbo Zhu

We introduce PMT2I, a novel Parallel Multilingual Prompting method for Text-to-Image tasks, which enhances multimodal LLM performance by leveraging multilingualism.
