FSE 2026

TestTailor: Generating High-Coverage Tests via Path-Proximal Tests with LLMs

Xiaoxuan Zhou, Yiling Lou, Jinhao Dong, Dan Hao

TestTailor transforms vague coverage goals into precise path-level guidance for LLM-based test generation by combining path-proximal test analysis with symbolic constraints from uncovered paths. On the CODAMOSA benchmark, it achieves up to 12.78% higher coverage while reducing API cost by ~60%.
